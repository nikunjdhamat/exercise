## Follow the below steps to run the app

## Step 1: Install the node modules

```bash
# using npm
npm install

# for ios do pod install
npm run pod-install

# OR using Yarn
yarn install

```

## Step 2: Start your Application

### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
